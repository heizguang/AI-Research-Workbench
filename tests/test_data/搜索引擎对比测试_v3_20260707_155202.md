**关键发现**：

1. **Tavily**：content 约 1500 字符，raw_content 约 2500 字符，是最佳选择
2. **Exa**：语义搜索，支持时间过滤，适合技术内容
3. **last30days**：多源聚合，覆盖 Reddit、X、YouTube、HN，适合获取最新动态
4. **传统搜索引擎**：百度有反爬虫限制，搜狗和 360 的 snippet 较短
